任务一

22281089 陈可致

一、实验目的

编写程序实现一个贝叶斯二值分类器，然后再实现一个多值分类器，网上找到一个
开源的文本分类数据集，进行测试，要求采用正确的评测方法，并在实验报告中给出评测结
果。

二、实验环境

  • Ubuntu 24.04.2 LTS
  • Python 3.12.3
  • 数据集：20 Newsgroups 数据集

三、实验内容

本实验实现两个部分：

  1. 朴素贝叶斯二值分类器：选取两个类别进行分类
  2. 朴素贝叶斯多值分类器：选取五个类别进行多分类

四、实验方法

  1. 使用 TfidfVectorizer 对文本进行向量化，去除停用词，限制最大文档频率。
  2. 使用 scikit-learn 中的 MultinomialNB 构建朴素贝叶斯模型。
  3. 使用 准确率、召回率、F1 值 评估模型性能。

七、实验总结

  • 朴素贝叶斯分类器在小规模文本分类任务中表现良好，尤其是类之间区分明显时。
  • 二值分类器的准确率较高，两个类别差异明显。
  • 多值分类中，不同类别的召回率差异较大，说明有的类容易混淆。
