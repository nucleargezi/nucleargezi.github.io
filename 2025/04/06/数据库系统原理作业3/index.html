task1

按如下格式要求在实验报告中描述所有涉及到的表的结构

用户表 (user)

表名 数据库用户 主键 外键 排序字段 索引字段
user root user_id 无 user_id user_name, phone, last_visit

字段信息

字段名称 数据类型 允许为空 唯一 默认值 约束条件
user_id BIGINT UNSIGNED N Y 主键
user_name VARCHAR(50) N Y
gender ENUM(‘男’, ‘女’) N N 男
phone VARCHAR(20) N Y
avatar_url VARCHAR(512) Y N ‘’
registered DATETIME N N CURRENT_TIMESTAMP
last_visit DATETIME N N CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP

朋友圈表 (moment)

表名 数据库用户 主键 外键 排序字段 索引字段
moment root moment_id user_id moment_id post_time, user_id, text_content

字段信息

字段名称 数据类型 允许为空 唯一 默认值 约束条件
moment_id BIGINT UNSIGNED N Y 主键
user_id BIGINT UNSIGNED N N 外键引用 user(user_id)
content_type ENUM(‘text’,’image’,’video’,’link’) N N text
text_content TEXT Y N
post_time DATETIME N N CURRENT_TIMESTAMP
like_count INT UNSIGNED Y N 0
comment_count INT UNSIGNED Y N 0

多媒体表 (media)

表名 数据库用户 主键 外键 排序字段 索引字段
media root media_id moment_id media_id moment_id

字段信息

字段名称 数据类型 允许为空 唯一 默认值 约束条件
media_id BIGINT UNSIGNED N Y 主键
moment_id BIGINT UNSIGNED N N 外键引用 moment(moment_id)
file_type VARCHAR(20) N N
file_path VARCHAR(255) N N
thumbnail_path VARCHAR(255) N N
file_size INT UNSIGNED Y N
upload_time DATETIME Y N CURRENT_TIMESTAMP

评论表 (comments)

表名 数据库用户 主键 外键 排序字段 索引字段
comments root comment_id moment_id, user_id, parent_id comment_id moment_id, comment_time, parent_id

字段信息

字段名称 数据类型 允许为空 唯一 默认值 约束条件
comment_id BIGINT UNSIGNED N Y 主键
moment_id BIGINT UNSIGNED N N 外键引用 moment(moment_id)
user_id BIGINT UNSIGNED N N 外键引用 user(user_id)
content TEXT N N
comment_time DATETIME N N CURRENT_TIMESTAMP
parent_id BIGINT UNSIGNED Y N NULL 外键引用 comments(comment_id)
is_top TINYINT(1) Y N 0

点赞表 (undefined_like)

表名 数据库用户 主键 外键 排序字段 索引字段
undefined_like root like_id moment_id, user_id like_id moment_id, user_id, like_time

字段信息

字段名称 数据类型 允许为空 唯一 默认值 约束条件
like_id BIGINT UNSIGNED N Y 主键
moment_id BIGINT UNSIGNED N N 外键引用 moment(moment_id)
user_id BIGINT UNSIGNED N N 外键引用 user(user_id)
like_time DATETIME N N CURRENT_TIMESTAMP

好友关系表 (friendship)

表名 数据库用户 主键 外键 排序字段 索引字段
friendship root friendship_id F_user, T_user friendship_id F_user, T_user, status

字段信息

字段名称 数据类型 允许为空 唯一 默认值 约束条件
friendship_id BIGINT UNSIGNED N Y 主键
F_user BIGINT UNSIGNED N N 外键引用 user(user_id)
T_user BIGINT UNSIGNED N N 外键引用 user(user_id)
status ENUM(‘pending’,’accepted’,’rejected’,’blocked’) N N pending
create_time DATETIME N N CURRENT_TIMESTAMP
